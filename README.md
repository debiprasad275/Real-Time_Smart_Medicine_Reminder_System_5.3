# Real-Time_Smart_Medicine_Reminder_System_5.3
An automated medicine dispensing system built using Raspberry Pi.
It dispenses medicine on schedule, provides water, and alerts the user using audio and display.

🚀 Features
⏰ Scheduled medicine dispensing

💊 Automatic compartment rotation (Servo)
🚰 Water dispensing system (DC pump)
🔊 Voice alerts via speaker
📟 OLED display for status updates
🌡️ Temperature & Humidity monitoring (DHT22)
🌐 Web interface (Flask) to manage medicines & schedules
